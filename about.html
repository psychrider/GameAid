Description
GametoGrow! is the first-ever modern platform which helps dyslexic and dyspraxic kids to overcome motor skills problems along with progress evaluation.
Features
  • First-Ever gamehub for dyslexic progress evaluation.
  • Games demanding body motion to help physical body.
  • Interactive website.
  • Custom designed website to have personal accountto judge perfomance time to time.
